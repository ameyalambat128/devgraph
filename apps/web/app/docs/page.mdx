# DevGraph Docs (MDX)

This site runs on the Next.js App Router with MDX enabled via `@next/mdx`.

## What is DevGraph?

- Parses `devgraph-*` fenced blocks in Markdown (service/api/env)
- Builds a project graph and emits `.devgraph/graph.json`
- Generates `summary.md`, per-service `AGENTS.md`, and Mermaid diagrams (`system.mmd` + optional `system.png`)

## CLI (MVP)

- Validate: `pnpm devgraph validate examples/*.md`
- Build: `pnpm devgraph build examples/*.md`
- Build with diff: `pnpm devgraph build examples/*.md --compare .devgraph/graph.json`
- Outputs land in `.devgraph/`

## Development

- Install deps: `pnpm install`
- Dev server: `pnpm dev --filter web`
- Build: `pnpm build --filter web`
- Lint: `pnpm lint --filter web`
